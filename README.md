# ぱくぱくフレンズ 🐣

4歳児向けのスマホ知育ゲーム。モンスター「ぱっくん」のお世話を通して、
毎日の生活習慣(ごはん・はみがき・おふろ・おかたづけ・ねんね)を楽しく学びます。

## 企画のねらい

| 項目 | 内容 |
|---|---|
| 対象 | 4歳前後(未就学児) |
| 学び | 生活習慣の順序・因果関係(お世話すると喜ぶ)・指先の巧緻性・1〜3の数 |
| 操作 | タップ・なでる・こする、のみ。文字が読めなくてもOK(絵文字アイコンで直感操作) |
| 安全設計 | 失敗・減点なし / 時間制限なし / 広告・課金なし / 外部通信なし |

## 遊び方

1. ホームでお世話を選ぶ
2. 🍚 **ごはん** — たべものをタップして「いち、に、さん」で食べさせる
3. 🪥 **はみがき** — 歯を指でこすって ばいきん をやっつける
4. 🛁 **おふろ** — 体をなでて あわあわ にして、シャワーで流す
5. 🧸 **おかたづけ** — 散らかったおもちゃをタッチして箱にしまう
6. 🌙 **ねんね** — 電気を消して寝かせ、おひさまをタップで朝に
7. 5つ全部でスタンプが揃うと 🎉 1日クリア!どうぶつのおともだちが1匹なかまに
8. 📖 **ずかん** — 集まったおともだちがいつでも見られる(全12匹)

## 起動方法

```bash
cd pakupaku-friends
python3 -m http.server 8000
# スマホ or ブラウザで http://localhost:8000 を開く
```

単一の `index.html` のみで動作します(依存なし)。
